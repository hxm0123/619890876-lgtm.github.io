## 2. 项目内容

本项目基于论文《Exploring the Controlling Factors of Watershed Streamflow Variability Using Hydrological and Machine Learning Models》，聚焦于滦河流域径流变化的主控因子分析。

*   **核心方法**：利用数据库流域模拟数据，基于R语言，将XGBoost与RF模型对径流量的预测值与实际值进行拟合分析。通过四个统计学指标(R^2、NSE、KGE、PBIAS）的量化对比，证明了XGBoost模型在整体预测性能上优于RF模型
*   **代码仓库**：[619890876-lgtm.github.io](https://github.com/619890876-lgtm/619890876-lgtm.github.io)
*   **研究方向**：水文水资源 / 流域径流变化模拟

## 3. 研究背景

1. 随着机器学习技术在水文学领域的广泛应用，诸如随机森林、梯度提升树等复杂模型在径流预测精度上往往优于传统物理模型。然而，这些“黑箱”模型缺乏物理机制的透明度，导致水文学家难以理解模型是如何做出决策的，限制了其在实际水资源管理中的信任度与应用深度。

   为了解决这一问题，SHAP（SHapley Additive exPlanations）方法被引入水文领域，用于量化各输入特征（如降雨、蒸散发、土壤湿度等）对模型输出的贡献度。特别是针对极端水文事件（如高径流洪水或低径流干旱），理解模型在特定时间步长的预测逻辑至关重要。本研究基于 `shapviz` 包，旨在通过可视化的“力图”，深入解析模型在特定子流域极端径流条件下的预测归因，从而增强机器学习模型在水文过程中的可解释性。



## 4. 本项目复现目标

本次复现的主要目标是重现原研究中关于**模型局部解释性分析**的关键图表，具体包括：

. **构建 SHAP 可视化对象**：利用训练好的水文机器学习模型，计算测试集数据的 SHAP 值，并封装为 `shapviz` 对象。
. **识别关键样本点**：筛选出具有代表性的极端水文时刻，具体关注**低径流**和**高径流**的典型样本（对应代码中的 `min_idx` 和 `max_idx`）。
. **生成特征贡献力图**：复现原文中的“图 7c”，即针对特定子流域的 SHAP 力图。该图需清晰展示各环境因子（如前期降雨、气温等）是正向推动还是负向抑制了径流的预测值，从而验证模型是否符合水文学常识（例如：降雨增加应正向贡献于径流）。

## 5.复现说明

### 5.1 运行环境
- **编程语言**：R (Version 4.x)
- **核心依赖包**：
    - `shapviz`：用于 SHAP 值的计算与高级可视化。
    - `ggplot2`：用于图形的二次美化与排版。
    - `xgboost` / `randomForest`：作为底层的机器学习预测模型（视具体原论文模型而定）。

### 5.2 关键步骤与逻辑
. **数据准备**：加载预处理后的子流域水文数据集，分离特征变量（X）与目标变量（y）。
. **SHAP 值计算**：
    - 使用 `shapviz()` 函数处理模型预测结果。
        - 代码逻辑：`sv <- shapviz(model, X_train, X_test)`
      . **极值索引定位**：
        - 通过排序找到径流预测值最大（洪水）和最小（枯水）的样本索引。
        - 代码逻辑：`max_idx <- which.max(predictions)`
      . **力图绘制**：
        - 调用 `sv_force()` 函数绘制单个样本的解释图。
        - 设置参数 `max_display` 控制显示的Top特征数量，确保图表清晰。
        - 使用 `labs()` 添加中文标题，说明子流域编号及对应的时间点。

## 6.预期结果

成功运行代码后，应生成两张关键的 SHAP 力图：

- **低径流力图**：展示在干旱或少雨时段，哪些特征（如低降雨量、高蒸发）导致了模型输出较低的径流值（通常表现为红色特征条向左推，蓝色向右推，最终汇聚于基础值 Base Value）。
- **高径流力图**：展示在暴雨或融雪时段，主要驱动因子（如高强度降雨）如何显著拉高预测值。
